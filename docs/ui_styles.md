# 《拾瓣传奇》界面设计风格文档 (UI Style Guide)

本文档定义了《拾瓣传奇》项目的界面设计规范，旨在确立统一、沉浸式的二次元 RPG 视觉风格。

## 1. 核心设计理念

*   **沉浸感 (Immersion)**：采用全屏布局与动态背景，弱化传统列表与表格，强化角色立绘展示。
*   **现代 RPG 感 (Modern RPG UX)**：借鉴现代主流二次元游戏（如《原神》、《崩坏：星穹铁道》）的布局逻辑。
*   **黑金极简 (Dark & Gold)**：以深色调为底，配合金色高亮，营造高级感与神秘感。

## 2. 色彩规范 (Color Palette)

### 2.1 基础色调
*   **主背景色**：`#0c0c0e` (深墨黑)
*   **面板背景**：`rgba(20, 20, 22, 0.7)` (带 15px 毛玻璃模糊)
*   **主文字色**：`#ececec` (浅灰白)
*   **次要文字**：`#888888` (中灰)

### 2.2 品牌与强调色
*   **高亮/金色**：`#f7d358` (用于等级、进度条、重要按钮)
*   **成功/绿色**：`#48bb78` (用于预览成功、属性提升)
*   **警告/橙色**：`#f6ad55` (用于主属性、强化提示)

### 2.3 稀有度颜色
*   **5★ (传说)**：`#FFD700` (金色)
*   **4★ (史诗)**：`#9B59B6` (紫色)
*   **3★ (稀有)**：`#409EFF` (蓝色)
*   **2★ (优秀)**：`#67C23A` (绿色)
*   **1★ (普通)**：`#909399` (灰色)

## 3. 字体规范 (Typography)

*   **字体家族**：`"PingFang SC", "Microsoft YaHei", "Source Han Sans CN", sans-serif`
*   **字号层级**：
    *   **角色大名**：`3rem` (48px)，带 `2px` 字间距与投影。
    *   **二级标题**：`1.5rem` (24px)。
    *   **正文/描述**：`1rem` (16px)，行高 `1.6`。
    *   **标签/注脚**：`0.75rem` (12px)。

## 4. 组件规范 (Components)

### 4.1 布局结构 (Layout)
*   **侧边导航 (Sidebar)**：宽度 `200px`，半透明背景，左侧 `4px` 激活条。
*   **主展示区 (Main Display)**：居中放置角色立绘，带有 `drop-shadow` 增强空间感。
*   **右侧信息面板 (Info Panel)**：宽度 `380px`，集成 Tabs 切换属性、技能与装备。

### 4.2 技能卡片 (Skill Cards)
*   背景色：`rgba(255, 255, 255, 0.03)`
*   边框：`1px solid rgba(255, 255, 255, 0.1)`
*   圆角：`8px`
*   **技能标签颜色**：
    *   普攻：`#4a5568`
    *   战技：`#2c5282`
    *   爆发：`#742a2a` (带红色呼吸感投影)

### 4.3 圣遗物与物品槽位 (Relic & Item Slots)
*   采用网格/列表混合布局。
*   图标展示：居中放置，保持 `80%` 填充率，支持物品自定义图标与占位符切换。
*   稀有度边框：根据稀有度显示不同背景色深度（`22` 或 `44` 透明度）与边框色。
*   等级标识：右下角（或左上角）黑色小标签。

## 5. 视觉特效 (Visual Effects)

*   **背景渐变**：`radial-gradient(circle at 70% 30%, rgba(45, 55, 72, 0.4) 0%, transparent 70%)`，模拟光源效果。
*   **毛玻璃 (Glassmorphism)**：`backdrop-filter: blur(10px)`。
*   **转场动画**：立绘切换使用 `fade` 动画（`0.3s`），增强流畅度。
*   **发光效果**：重要数值或进度条使用 `box-shadow` 产生微光感。

## 6. 交互准则

*   **反馈**：所有按钮点击必须触发 `audio/click.mp3`。
*   **响应式**：当前设计主要针对 `16:9` 及以上宽屏横屏环境。
*   **空状态**：使用统一的 `icons.empty` 提示图片。
