# 角色技能设计基本准则

本指南旨在为《十班全明星》的角色设计提供统一的逻辑框架，确保新角色的机制既有趣又符合游戏整体平衡。

## 1. 核心设计哲学
*   **差异化**：每个角色应至少有一个独特的机制（如光环、松子、生命值消耗、溢出转化等）。
*   **逻辑自洽**：角色的背景描述、属性类型与技能效果应保持一致。
*   **风险与收益平衡**：高收益技能（高倍率、强控）应伴随相应的代价（高能耗、扣血、自身减益）。

## 2. 技能结构
每个角色必须包含三个核心技能，其设计应遵循以下逻辑：

### 普攻 (General)
*   **定位**：主要用于回复战技点或积攒核心机制。
*   **倍率标准**：100% ATK。
*   **设计建议**：除非是辅助或坦克，否则不建议在此技能中加入过强的特效。

### 战技 (Skill)
*   **定位**：角色的核心输出或功能手段。
*   **能耗**：通常消耗 1 点战技点。
*   **倍率标准**：200% ATK。
*   **特殊效果**：
    *   **输出型**：可加入无视防御、生命百分比加成等。
    *   **功能型**：治疗、净化、单体增益/减益。

### 爆发技 (Super Skill)
*   **定位**：扭转战局的强力手段。
*   **能耗**：消耗 3 点战技点。
*   **倍率标准**：400% ATK (单体) 或 250% ATK (群体)。
*   **设计要求**：视觉效果和数值上都要有明显的压制感，通常包含多回合的持续效果。

## 3. 属性设计建议
*   **火 (Fire)**：侧重持续伤害、爆发、攻击力提升。
*   **水 (Water)**：侧重治疗、生命值加成、减伤。
*   **雷 (Thunder)**：侧重速度、多段攻击、能量回复。
*   **草 (Grass)**：侧重控制、干扰、周期性效果。
*   **量子 (LiangZi)**：侧重无视防御、位移、机制拆解。
*   **虚无 (Nihility)**：侧重负面状态、属性削弱、延迟爆发。
*   **物理 (Physics)**：侧重纯粹数值、击破、稳定性。

## 4. 注意事项
*   **避免数值膨胀**：设计倍率时应参考 `base.ts` 中的标准值。
*   **描述清晰**：技能描述应准确反映代码实现（如：“造成 100% 攻击力的伤害”而非“造成 100 伤害”）。
*   **内部名称一致**：`inside_name` 必须与代码类名保持一致，以便加载系统识别。
