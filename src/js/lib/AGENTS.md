# src/js/lib/ - 工具库

**生成时间：** 2026-01-22
**提交：** 332e8c2
**分支：** main

## 概述
专用工具库，用于外部 API 集成（AI、CAPTCHA）和复杂数据生成。

## 结构
```
./
├── ai.ts       # 百度千帆 AI 聊天集成，带对话历史
├── name.ts     # 复杂随机名称生成器
└── gt4.js      # 极验 v4 CAPTCHA 提供商
```

## 反模式
- **过度工程化的生成**：`name.ts` 使用 696 行和约 500 个静态数组条目进行随机名称生成，该任务可以用少量代码实现。

## 约定
- **外部 API 包装器**：每个模块是特定外部服务的自包含包装器
- **多 CDN 回退**：`gt4.js` 实现了稳健的回退链（npmmirror、ghproxy、GitHub）用于加载极验库，这是弹性外部依赖的模式
- **混淆凭证**：`ai.ts` 模块使用双重 base64 编码进行 API 凭证处理，这是一种非标准的"通过混淆实现安全"的方法
